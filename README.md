## サイト概要
### サイトテーマ
恋人や家族と行きたい場所を優先度（行きたい順）ごとにメモしておけるようなアプリ

### テーマを選んだ理由
インスタとかで行きたい場所をメモすると、多すぎて結局マップにピンだらけでよくわからなくなることがよくあるから。

### ターゲットユーザー
旅行・カフェ巡りが好きな方、インスタのマップがピンだらけでぐちゃぐちゃな方
